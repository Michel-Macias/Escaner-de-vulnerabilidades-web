# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Este proyecto se distribuye con fines educativos y de auditoría ética. Si encuentras una mejora de seguridad relevante para el propio scanner (no para un target real), abre un issue etiquetado como `security` o envía un PR con tu hallazgo y la mitigación propuesta.

No reportes vulnerabilidades encontradas en terceros sin autorización escrita expresa del propietario del sistema.
